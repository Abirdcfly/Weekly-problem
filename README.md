问题1：

给你一串包含'X'和'O'字符串，如"OXOOOXXXOXXOOOXXXOXO",问这个字符串中，与其中一个X直接相连的O最多有多少个？比如"OXOO",就有三个O与X相连。但是"OXOOOXOO"最多就是五个而不是六个，因为左边的X挡住了最左边的O。

	输入：字符串 输出：整数
	
	例子：
	输入:"OXO"			输出：2
	输入:"OXOOXOXXOOOXO"	输出：3
	输入:"OOXOOOXOXO"		输出：5
	输入:"OOOOOOOOOO"	 	输出：0
	输入:"XXX"			输出：0
